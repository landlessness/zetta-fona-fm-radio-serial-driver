##Zetta FONA FM Radio serial device driver

###Install

```
$> npm install zetta-fona-fm-radio-serial-driver
```

###Usage

```
var zetta = require('zetta');
var FonaFMRadio = require('zetta-fona-fm-radio-serial-driver');

zetta()
  .use(FonaFMRadio)
  .listen(1337)
```
